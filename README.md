## Dataset

The dataset used in this project is the [BBC Text Dataset](https://www.kaggle.com/datasets/rtatman/bbc-news-articles), which comprises 2,225 articles spanning several categories. 

### Categories Included
- **Technology**
- **Business**
- **Sports**
- **Entertainment**
- **Politics**
